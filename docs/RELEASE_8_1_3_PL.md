# Matrix Energy Center 8.1.3

W szufladzie powiadomień kiosku, pod ostatnim komunikatem, dodano przycisk
**Usuń powiadomienia z listy**.

Przycisk:

- usuwa odczytane wpisy z listy bieżącego profilu kiosku,
- zamyka zwykłe aktywne komunikaty także w Matrix Notification Center,
- zapamiętuje wyczyszczenie po przeładowaniu strony,
- nie usuwa aktywnych komunikatów wymagających potwierdzenia.

Do działania mostu wymagany jest Matrix Notification Center 1.5.1 lub nowszy.
Po aktualizacji uruchom ponownie Home Assistant i odśwież pamięć strony w Fully
Kiosk.
